# Complexity of various operations on:
- Binary search tree
- Hash Table w/ linear probing

# Search
- BST: O(n)
- Hash Table: O(n)

# Insert
- BST: O(n)
- Hash Table: O(n)

# Delete
- BST: O(n)
- Hash Table: O(n)

# Sort
- BST: O(n)
- Hash Table: O(nlog(n))

# Range
- BST: O(n)
- Hash Table: O(n)
